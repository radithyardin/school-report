# School Report

Aplikasi School Report adalah sistem pelaporan sekolah berbasis web.

## Struktur Folder

```
school-report/
├── frontend/    # Aplikasi React (Frontend)
└── backend/     # API PHP (Backend)
```

## Cara Install & Jalankan

### Frontend
```bash
cd frontend
npm install
npm run dev
```

### Backend
1. Install XAMPP (PHP + MySQL)
2. Import `backend/api/database.sql` ke phpMyAdmin
3. Jalankan di localhost
