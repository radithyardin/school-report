# School Report - Frontend

Frontend aplikasi School Report menggunakan React + Vite + Tailwind CSS.

## Cara Install & Jalankan

```bash
cd frontend
npm install
npm run dev
```

## Build untuk Production

```bash
npm run build
npm run preview
```

## Struktur Folder

```
frontend/
├── src/
│   ├── api/          # Konfigurasi API
│   ├── assets/        # Gambar, icon, dll
│   ├── components/   # Komponen reusable (Navbar, Footer, dll)
│   ├── layout/       # Layout (AdminLayout, UserLayout)
│   └── pages/        # Halaman aplikasi
│       ├── admin/    # Halaman admin
│       └── user/     # Halaman user
├── public/           # File statis
└── index.html        # Entry point
```
